#  Loja de Cosméticos API

API REST para gerenciamento de uma loja de cosméticos, com cadastro de clientes, produtos, categorias, pedidos e itens de pedido.

## Tecnologias

- Java 22 · Spring Boot 3.5 · Spring Data JPA · MySQL · Lombok · Maven

## Endpoints

| Recurso       | Rota            |
|---------------|-----------------|
| Clientes      | /cliente        |
| Categorias    | /categoria      |
| Produtos      | /produto        |
| Pedidos       | /pedido         |
| Itens Pedido  | /item-pedido    |
